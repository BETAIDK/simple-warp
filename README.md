Simple warp is a simple warping system providing owners of minecraft
server to easily create and manage warps.

## Commands

| Command       | Usage         |
| ------------- | ------------- |
| Warp          |     /Warp     |
| SetWarp       |     /Setwarp  |
| UnWarp       |     /Unwarp  |
| WarpList       |     /Warplist  |
